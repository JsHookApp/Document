# Connect to Computer

You can use `Visual Studio Code` to push scripts to your phone in real-time, or pull scripts from your phone to your computer.

Download the VS Code extension here: [https://marketplace.visualstudio.com/items?itemName=JsonET.jshook-vscode-extension](https://marketplace.visualstudio.com/items?itemName=JsonET.jshook-vscode-extension)

After installation, right-click on a `js` file or use keyboard shortcuts:

`Shift+Alt+D`: Push script

`Shift+Alt+C`: Pull script

`Shift+Alt+F`: Clear logs

`Shift+Alt+G`: View logs (auto-refresh)

When syncing scripts, the (script name + relative path) must match what's on the phone. When using the extension, enable the server in JsHook settings. After pressing the shortcut, an IP input dialog will appear — make sure your computer and phone are on the same Wi-Fi network.

**Enabling the server does not require JsHook to stay in the background.**

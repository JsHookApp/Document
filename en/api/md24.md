# r2 — r2frida Reverse Engineering Engine

Based on [r2frida](https://github.com/nowsecure/r2frida), provides disassembly, memory search, module analysis, function tracing, interception and other low-level reverse engineering capabilities.
All methods execute directly in the Frida GumJS runtime without Java layer bridging.

---

## Disassembly

### `r2.disasm(address, count)`

Disassemble machine code instructions at the specified address.

| Parameter | Type | Description |
|-----------|------|-------------|
| address | `string` / `NativePointer` | Start address |
| count | `number` | Number of instructions, default 32 |

**Returns**: `string` — Disassembly text with addresses, mnemonics, operands and symbol info.

```javascript
var base = Module.findBaseAddress("libil2cpp.so");
var asm = r2.disasm(base.add(0x1234), 10);
console.log(asm);
```

## Memory

### `r2.hexdump(address, size)`

Dump memory contents in hexadecimal format.

### `r2.memoryRanges()`

List all memory ranges of the process.

### `r2.memoryRangesJson()`

Return memory ranges in JSON format.

### `r2.memoryRangesHere(address)`

Find the memory range containing the specified address.

### `r2.memoryMaps()`

List memory maps (merged consecutive ranges with same path).

### `r2.allocate(size)`

Allocate a memory block of the specified size on the heap.

### `r2.allocateString(str)`

Allocate a UTF-8 string on the heap.

### `r2.freeAlloc(addr)`

Free memory previously allocated via `allocate` / `allocateString`.

### `r2.listAllocs()`

List all memory addresses allocated via r2.

### `r2.changeProtection(addr, size, perm)`

Change memory region protection attributes.

| Parameter | Type | Description |
|-----------|------|-------------|
| addr | `string` | Memory address |
| size | `number` | Size in bytes |
| perm | `string` | Permission string, e.g. `"rwx"`, `"r-x"` |

## Search

### `r2.searchString(pattern)`

Search for a string in readable memory ranges.

**Returns**: `string` (JSON) — Hit list `[{address, size}]`

```javascript
var hits = JSON.parse(r2.searchString("GameManager"));
hits.forEach(function(h) {
    console.log("Found at: " + h.address);
});
```

### `r2.searchHex(hex)`

Search for a hexadecimal pattern.

### `r2.searchValue4(value)`

Search for a 4-byte (32-bit little-endian) value.

### `r2.searchValue8(value)`

Search for an 8-byte (64-bit little-endian) value.

### `r2.searchWide(pattern)`

Search for a UTF-16 wide string.

### `r2.searchStrings(minLen)`

Scan memory for all printable strings.

## Trace

### `r2.trace(address)`

Trace function calls. Automatically prints call stack, arguments and return values. Supports `java:` prefix for Java methods.

```javascript
r2.trace(base.add(0x1234));
r2.trace("java:com.example.App.checkLicense");
```

### `r2.listTraces()` / `r2.clearTrace(address)` / `r2.clearAllTraces()`

Manage active traces.

---

## Intercept

### `r2.interceptRet0(target)` / `r2.interceptRet1(target)`

Intercept function and force return 0 or 1. Supports `java:` prefix.

### `r2.interceptRetInt(target, value)`

Intercept and return specific integer.

### `r2.interceptRetString(target, str)`

Intercept and return specific string.

### `r2.interceptDetachAll()`

Remove all intercepts.

---

## Stalker

### `r2.stalkFunction(address, timeout)`

Instruction-level function tracing using Frida Stalker (experimental).

**Returns**: `Promise<string>` — Call summary (count × symbol name)

## General

### `r2.info()`

Return process info as JSON string.

### `r2.version()`

Return r2frida version.

### `r2.cmd(command)`

Execute r2-style command string.

```javascript
r2.cmd("pd 0x71a2b234 20");  // disassemble
r2.cmd("il");                  // module list
r2.cmd("/x 48 89 5c 24 08");  // hex search
r2.cmd("dm");                  // memory maps
```

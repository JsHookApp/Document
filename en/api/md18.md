# kernel

APIs for kernel mode. Requires the fridamod framework. Use `runtime.isKernel` to check if running in kernel mode.

In kernel mode, the target app is not injected. Instead, a spectator process (clone) is created, and Frida runs within it. Only JsHook-provided APIs are available — hook-related APIs are disabled.

## kernel.getModuleBase(libname)

Get the memory base address of a shared library, useful for memory offset calculations.

`Parameters`:

`libname`: string

`Returns`: string

## kernel.readDword(address)

Read an integer value from a memory address.

`Parameters`:

`address`: string

`Returns`: long

## kernel.readFloat(address)

Read a float value from a memory address.

`Parameters`:

`address`: string

`Returns`: float

## kernel.writeDword(address,value)

Write an integer value to a memory address.

`Parameters`:

`address`: string

`value`: long

`Returns`: int

## kernel.writeFloat(address,value)

Write a float value to a memory address.

`Parameters`:

`address`: string

`value`: float

`Returns`: int

## Example Script

Download the practice target app: [https://github.com/JsHookApp/Download/releases/download/files/Learn.Frida.apk](https://github.com/JsHookApp/Download/releases/download/files/Learn.Frida.apk)

And the corresponding `dump.cs` file: [https://github.com/JsHookApp/Download/releases/download/files/Learn.Frida_dump.cs](https://github.com/JsHookApp/Download/releases/download/files/Learn.Frida_dump.cs)

Analyze `public class FieldTest : MonoBehaviour` in the `Learn.Frida_dump.cs` file.

The script below first finds the memory base address using non-kernel mode Frida, then modifies variables in kernel mode. Click the `ShowResult` button to get an input box for searching the current `staticA` value in memory.

```javascript
if (runtime.isKernel) {
    console.log('kernel start')
    const time = setInterval(() => {
        const il2cpp = kernel.getModuleBase('libil2cpp.so')
        if (il2cpp) {
            clearInterval(time)
            console.log('il2cpp', il2cpp)
            setInterval(() => {
                const newval = Math.floor(Math.random() * 900) + 100
                kernel.writeDword('0x7e3fe30c30', newval)
                const staticA = kernel.readDword('0x7e3fe30c30')
                console.log('staticA', staticA)
            }, 3000)
        }
    }, 100)
}
```

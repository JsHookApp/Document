# kernel

APIs used in kernel mode. These require the fridamod framework. Use `runtime.isKernel` to check whether the script is currently running in kernel mode.

In kernel mode, the target app is **not** injected directly. Instead, a separate observer process is cloned, and Frida runs inside that clone. Within the script you may only use the APIs that JsHook provides — the regular hooking APIs have no effect here.

## kernel.getModuleBase(libname)

Returns the in-memory base address of a shared library (`.so`), which you can use for memory offset calculations.

`Parameters`:

`libname`: string

`Returns`: string

## kernel.readDword(address)

Reads an integer value from a memory address.

`Parameters`:

`address`: string

`Returns`: long

## kernel.readFloat(address)

Reads a float value from a memory address.

`Parameters`:

`address`: string

`Returns`: float

## kernel.writeDword(address,value)

Writes an integer value to a memory address.

`Parameters`:

`address`: string

`value`: long

`Returns`: int

## kernel.writeFloat(address,value)

Writes a float value to a memory address.

`Parameters`:

`address`: string

`value`: float

`Returns`: int

## Example Script

First download the practice target app: [https://github.com/JsHookApp/Download/releases/download/files/Learn.Frida.apk](https://github.com/JsHookApp/Download/releases/download/files/Learn.Frida.apk)

And the matching `dump.cs` file: [https://github.com/JsHookApp/Download/releases/download/files/Learn.Frida_dump.cs](https://github.com/JsHookApp/Download/releases/download/files/Learn.Frida_dump.cs)

In `Learn.Frida_dump.cs`, the methods to focus on are those of `public class FieldTest : MonoBehaviour`.

In the example below, the script first locates the memory base address using Frida in non-kernel mode (you are free to find the base address any other way), and then modifies the variable's value in kernel mode. Tapping the `ShowResult` button brings up an input box where you enter the current value of `staticA` to search for its memory address.

```javascript
// Memory scan
const memoryScan = (m, pattern, call) => {
    Memory.scan(m.base, m.size, pattern, {
        onMatch(address, size) {
            call(address)
            return 'stop';
        },
        onComplete() {
        }
    });
}

function getRandomInt(min, max) {
    return Math.floor(Math.random() * (max - min + 1)) + min;
}

// Start using the kernel
if (runtime.isKernel) {

    console.log('kernel start')

    const time = setInterval(() => {
        const il2cpp = kernel.getModuleBase('libil2cpp.so')
        if (il2cpp) {
            clearInterval(time)
            console.log('il2cpp', il2cpp)

            // Read / modify the value
            setInterval(() => {
                const newval = getRandomInt(100, 999)
                console.log('newval', newval)
                kernel.writeDword('0x7e3fe30c30', newval)

                const kernel_staticA = kernel.readDword('0x7e3fe30c30')
                console.log('staticA', kernel_staticA)
            }, 3000)
        }
    }, 100)
} else {

    // In non-kernel mode, use Frida to find the variable's memory base address.
    // You can also obtain it through other means.

    const time = setInterval(() => {
        const addr = Module.findBaseAddress('libil2cpp.so')
        if (addr) {
            clearInterval(time)
            console.log('libil2cpp ok')
            // The il2cpp module address is dynamic
            const il2cpp = ptr(addr)
            console.log('il2cpp', il2cpp)
            // 0x667F14 is a fixed offset pointing to the ShowResult method
            Interceptor.attach(il2cpp.add(0x667F14), {
                onEnter: function (args) {
                    // The current instance
                    const instance = args[0]

                    // Enter the current value of staticA to search memory for it
                    dialog.input('search_val', {
                        ok: function (res) {

                            // Get the search range
                            const range = Process.findRangeByAddress(instance)

                            // Build the search pattern
                            const pattern = ptr(parseInt(res)).toMatchPattern().replace(' 00 00 00 00', '')

                            memoryScan(range, pattern, (address) => {
                                // The variable's memory address
                                console.log('staticA_address', address)

                                // The variable's value
                                const staticA = ptr(address).readInt()
                                console.log('staticA', staticA)

                                // Modify it
                                ptr(address).writeInt(1000)
                            })
                        },
                        cancel: function () {
                        }
                    }, '')
                },
                onLeave: function (retval) {
                }
            })
        }
    }, 100)

}
```

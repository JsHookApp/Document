# colors

Color string and value conversions.

## colors.toString(color)

Returns the string representation of a color value in "#AARRGGBB" format.

`Parameters`:

`color`: int — RGB color value

`Returns`: string

## colors.red(color)

Returns the R channel value of the color, range 0-255.

`Parameters`:

`color`: int — RGB color value

`Returns`: int

## colors.green(color)

Returns the G channel value of the color, range 0-255.

`Parameters`:

`color`: int — RGB color value

`Returns`: int

## colors.blue(color)

Returns the B channel value of the color, range 0-255.

`Parameters`:

`color`: int — RGB color value

`Returns`: int

## colors.alpha(color)

Returns the Alpha channel value of the color, range 0-255.

`Parameters`:

`color`: int — RGB color value

`Returns`: int

## colors.rgb(red,green,blue)

Returns the integer color value composed of these color channels.

`Parameters`:

`red`: int — R channel value

`green`: int — G channel value

`blue`: int — B channel value

`Returns`: int

## colors.argb(alpha,red,green,blue)

Returns the integer color value composed of these color channels.

`Parameters`:

`alpha`: int — Alpha channel value

`red`: int — R channel value

`green`: int — G channel value

`blue`: int — B channel value

`Returns`: int

## colors.parseColor(color)

Returns the integer color value from a color string.

`Parameters`:

`color`: string — Color string

`Returns`: int

# runtime

Get basic information about the current hook runtime.

## runtime.appInfo

`Returns`: ApplicationInfo

## runtime.packageName

`Returns`: string

## runtime.processName

`Returns`: string

## runtime.classLoader

`Returns`: ClassLoader

## runtime.isKernel

Whether running in kernel mode.

`Returns`: boolean

## runtime.coreVersionCode

Get the current JsHook version code.

`Returns`: int

## runtime.modVersion

Get the current fridamod version.

`Returns`: string

## runtime.modVersionCode

Get the current fridamod version code.

`Returns`: int

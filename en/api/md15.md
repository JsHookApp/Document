# keys

Simulate key operations. Requires the Magisk module to be installed.

## keys.click(x,y)

Simulate a tap at position (x, y).

`Parameters`:

`x`: int

`y`: int

## keys.swipe(x,y,tox,toy,time)

Simulate a swipe from (x, y) to (tox, toy) with a duration.

`Parameters`:

`x`: int

`y`: int

`tox`: int

`toy`: int

`time`: int

## keys.back()

Press back button.

## keys.home()

Press home button.

## keys.enter()

Press enter key.

## keys.del()

Press delete key.

## keys.text(text)

Simulate text input. Make sure the text input field is focused before calling this.

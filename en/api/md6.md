# device

Get basic device information.

## device.isAdbEnabled()

Check if ADB is enabled on the device.

`Returns`: boolean

## device.getSDKVersionName()

Get device system version name.

`Returns`: string

## device.getSDKVersionCode()

Get device system version code.

`Returns`: int

## device.getAndroidID()

Get device Android ID.

`Returns`: string

## device.getMacAddress()

Get device MAC address.

`Returns`: string

## device.getManufacturer()

Get device manufacturer.

`Returns`: string

## device.getModel()

Get device model.

`Returns`: string

## device.getABIs()

Get device ABIs.

`Returns`: string

## device.isTablet()

Check if the device is a tablet.

`Returns`: boolean

## device.isDevelopmentSettingsEnabled()

Check if developer options are enabled.

`Returns`: boolean

## device.getScreenWidth()

Get screen width in pixels.

`Returns`: int

## device.getScreenHeight()

Get screen height in pixels.

`Returns`: int

## device.getScreenDensity()

Get screen density.

`Returns`: float

## device.getScreenDensityDpi()

Get screen density DPI.

`Returns`: int

## device.isLandscape()

Check if the device is in landscape mode.

`Returns`: boolean

## device.isPortrait()

Check if the device is in portrait mode.

`Returns`: boolean

## device.screenShot(activity)

Take a screenshot.

`Parameters`:

`activity`: Activity

`Returns`: Bitmap

## device.setClipboard(data)

Set clipboard content.

`Parameters`:

`data`: string

## device.getClipboard()

Get clipboard content.

`Returns`: string

# dex

> **FridaMod framework only**

DexKit integration API for searching classes, methods, and fields in target app's DEX files. Powered by [DexKit](https://github.com/LuckyPray/DexKit) deep integration with high-performance native execution.

## dex.init(apkPath)

Initialize DexKit with an APK file path. When `apkPath` is null or empty, automatically uses the current app's APK path.

`Parameters`:

`apkPath`: string | null

`Returns`: boolean

## dex.initByClassLoader()

Initialize using the target app's ClassLoader (uses in-memory DEX data).

`Returns`: boolean

## dex.close()

Close and release DexKit resources. Must be called after use.

## dex.isValid()

Check if DexKit is initialized.

`Returns`: boolean

## dex.getDexCount()

Get the number of loaded DEX files.

`Returns`: int

## dex.findClassByStrings(usingStrings, searchPackages, excludePackages)

Search for classes containing specified strings.

`Parameters`:

`usingStrings`: string — JSON array format, e.g. `'["VIP","isExpired"]'`

`searchPackages`: string | null — Package name scope, comma-separated, e.g. `"com.example,com.test"`

`excludePackages`: string | null — Excluded package names, comma-separated

`Returns`: string — JSON array

```json
[{"name":"com.example.VipManager","superClass":"java.lang.Object","interfaces":[],"accessFlags":1}]
```

## dex.findClassByAnnotation(annotationType, searchPackages)

Search for classes by annotation type.

`Parameters`:

`annotationType`: string — Fully qualified annotation type name, e.g. `"com.example.MyAnnotation"`

`searchPackages`: string | null — Package name scope, comma-separated

`Returns`: string — JSON array

## dex.findClass(query)

Advanced multi-criteria class search.

`Parameters`:

`query`: string — JSON query conditions

```json
{
  "searchPackages": ["com.example"],
  "excludePackages": ["com.example.test"],
  "usingStrings": ["str1", "str2"],
  "superClass": "android.app.Activity",
  "interfaces": ["java.io.Serializable"],
  "methodCount": {"min": 1, "max": 10},
  "fieldCount": {"min": 0, "max": 5}
}
```

`Returns`: string — JSON array

## dex.findMethodByStrings(usingStrings, searchPackages)

Search for methods containing specified strings.

`Parameters`:

`usingStrings`: string — JSON array format, e.g. `'["str1","str2"]'`

`searchPackages`: string | null — Package name scope, comma-separated

`Returns`: string — JSON array

```json
[{"className":"com.example.Vip","name":"check","returnType":"boolean","paramTypes":[],"accessFlags":1,"descriptor":"Lcom/example/Vip;->check()Z"}]
```

## dex.findMethodBySignature(returnType, paramTypes, modifiers, searchPackages)

Search for methods by signature characteristics.

`Parameters`:

`returnType`: string | null — Return type, e.g. `"boolean"`, `"void"`

`paramTypes`: string | null — JSON array of parameter types, e.g. `'["int","java.lang.String"]'`

`modifiers`: string | null — Modifiers, e.g. `"public static"`

`searchPackages`: string | null — Package name scope, comma-separated

`Returns`: string — JSON array

## dex.findMethod(query)

Advanced multi-criteria method search.

`Parameters`:

`query`: string — JSON query conditions

```json
{
  "searchPackages": ["com.example"],
  "usingStrings": ["premium"],
  "returnType": "boolean",
  "paramTypes": ["int", "java.lang.String"],
  "paramCount": 2,
  "modifiers": "public",
  "methodName": "isVip",
  "className": "com.example.Vip",
  "usingNumbers": [1, 0.5, 114514]
}
```

`Returns`: string — JSON array

## dex.findField(query)

Search for fields.

`Parameters`:

`query`: string — JSON query conditions

```json
{
  "searchPackages": ["com.example"],
  "fieldType": "int",
  "fieldName": "count",
  "modifiers": "private static",
  "className": "com.example.Counter"
}
```

`Returns`: string — JSON array

```json
[{"className":"com.example.Counter","name":"count","type":"int","accessFlags":10,"descriptor":"Lcom/example/Counter;->count:I"}]
```

## dex.batchFindClassByStrings(queriesJson)

Batch search for classes by strings (high performance, single traversal for multiple queries).

`Parameters`:

`queriesJson`: string — JSON format `{"key1":["str1","str2"], "key2":["str3"]}`

`Returns`: string — JSON format `{"key1":[{classData},...], "key2":[{classData},...]}`

## dex.batchFindMethodByStrings(queriesJson)

Batch search for methods by strings (high performance, single traversal for multiple queries).

`Parameters`:

`queriesJson`: string — JSON format `{"key1":["str1","str2"], "key2":["str3"]}`

`Returns`: string — JSON format `{"key1":[{methodData},...], "key2":[{methodData},...]}`

## dex.getClassData(className)

Get detailed class information including fields and methods.

`Parameters`:

`className`: string — Fully qualified class name, e.g. `"com.example.MyClass"`

`Returns`: string — JSON format

```json
{
  "name": "com.example.MyClass",
  "superClass": "java.lang.Object",
  "interfaces": ["java.io.Serializable"],
  "accessFlags": 1,
  "fields": [{"className":"com.example.MyClass","name":"id","type":"int","accessFlags":2,"descriptor":"..."}],
  "methods": [{"className":"com.example.MyClass","name":"getId","returnType":"int","paramTypes":[],"accessFlags":1,"descriptor":"..."}]
}
```

## Example Scripts

### Basic Search

```javascript
// Initialize DexKit (auto-detect current app's APK)
dex.init(null);
console.log('DEX file count:', dex.getDexCount());

// Search classes containing "VIP" and "expired"
var classes = JSON.parse(dex.findClassByStrings('["VIP","expired"]', null, null));
for (var i = 0; i < classes.length; i++) {
    console.log('Found class:', classes[i].name);
}

// Release resources
dex.close();
```

### Search and Hook Methods

```javascript
dex.init(null);

// Search methods containing "premium" that return boolean
var methods = JSON.parse(dex.findMethod(JSON.stringify({
    usingStrings: ["premium"],
    returnType: "boolean",
    paramCount: 0
})));

for (var i = 0; i < methods.length; i++) {
    console.log('Found method:', methods[i].className + '.' + methods[i].name);
    
    // Hook the found method using Frida
    Java.perform(function() {
        var cls = Java.use(methods[i].className);
        cls[methods[i].name].implementation = function() {
            console.log('Method called, forcing return true');
            return true;
        };
    });
}

dex.close();
```

### Batch Search

```javascript
dex.init(null);

// Batch search multiple string groups (more efficient than multiple findClassByStrings calls)
var result = JSON.parse(dex.batchFindClassByStrings(JSON.stringify({
    "vip_class": ["VIP", "Premium"],
    "ad_class": ["AdMob", "Banner"],
    "pay_class": ["payment", "purchase"]
})));

for (var key in result) {
    console.log('[' + key + '] Found ' + result[key].length + ' classes:');
    for (var i = 0; i < result[key].length; i++) {
        console.log('  - ' + result[key][i].name);
    }
}

dex.close();
```

### Get Class Details

```javascript
dex.init(null);

var info = JSON.parse(dex.getClassData('com.example.VipManager'));
if (info.name) {
    console.log('Class:', info.name);
    console.log('Super:', info.superClass);
    console.log('Interfaces:', JSON.stringify(info.interfaces));
    console.log('Methods:', info.methods.length);
    for (var i = 0; i < info.methods.length; i++) {
        var m = info.methods[i];
        console.log('  Method:', m.name + '(' + m.paramTypes.join(',') + ') -> ' + m.returnType);
    }
}

dex.close();
```

# dex

> **仅 FridaMod 框架支持**

DexKit 集成 API，用于在目标应用中搜索类、方法、字段等 DEX 信息。基于 [DexKit](https://github.com/LuckyPray/DexKit) 深度集成，所有搜索操作在 Native 层高性能执行。

## dex.init(apkPath)

通过 APK 文件路径初始化 DexKit，`apkPath` 为空时自动获取当前应用的 APK 路径。

`参数`:

`apkPath`: string | null

`返回值`: boolean

## dex.initByClassLoader()

通过目标应用的 ClassLoader 初始化（使用内存中的 DEX 数据）。

`返回值`: boolean

## dex.close()

关闭并释放 DexKit 资源。使用完毕后务必调用。

## dex.isValid()

判断 DexKit 是否已初始化。

`返回值`: boolean

## dex.getDexCount()

获取已加载的 DEX 文件数量。

`返回值`: int

## dex.findClassByStrings(usingStrings, searchPackages, excludePackages)

按字符串搜索类。搜索 DEX 中包含指定字符串的类。

`参数`:

`usingStrings`: string — JSON 数组格式，如 `'["VIP","isExpired"]'`

`searchPackages`: string | null — 搜索包名范围，逗号分隔，如 `"com.example,com.test"`

`excludePackages`: string | null — 排除包名范围，逗号分隔

`返回值`: string — JSON 数组

```json
[{"name":"com.example.VipManager","superClass":"java.lang.Object","interfaces":[],"accessFlags":1}]
```

## dex.findClassByAnnotation(annotationType, searchPackages)

按注解类型搜索类。

`参数`:

`annotationType`: string — 注解类型全限定名，如 `"com.example.MyAnnotation"`

`searchPackages`: string | null — 搜索包名范围，逗号分隔

`返回值`: string — JSON 数组

## dex.findClass(query)

多条件搜索类（高级）。

`参数`:

`query`: string — JSON 查询条件

```json
{
  "searchPackages": ["com.example"],
  "excludePackages": ["com.example.test"],
  "usingStrings": ["str1", "str2"],
  "superClass": "android.app.Activity",
  "interfaces": ["java.io.Serializable"],
  "methodCount": {"min": 1, "max": 10},
  "fieldCount": {"min": 0, "max": 5}
}
```

`返回值`: string — JSON 数组

## dex.findMethodByStrings(usingStrings, searchPackages)

按字符串搜索方法。搜索 DEX 中包含指定字符串的方法。

`参数`:

`usingStrings`: string — JSON 数组格式，如 `'["str1","str2"]'`

`searchPackages`: string | null — 搜索包名范围，逗号分隔

`返回值`: string — JSON 数组

```json
[{"className":"com.example.Vip","name":"check","returnType":"boolean","paramTypes":[],"accessFlags":1,"descriptor":"Lcom/example/Vip;->check()Z"}]
```

## dex.findMethodBySignature(returnType, paramTypes, modifiers, searchPackages)

按方法签名特征搜索方法。

`参数`:

`returnType`: string | null — 返回类型，如 `"boolean"`、`"void"`

`paramTypes`: string | null — JSON 数组格式的参数类型列表，如 `'["int","java.lang.String"]'`

`modifiers`: string | null — 修饰符，如 `"public static"`

`searchPackages`: string | null — 搜索包名范围，逗号分隔

`返回值`: string — JSON 数组

## dex.findMethod(query)

多条件搜索方法（高级）。

`参数`:

`query`: string — JSON 查询条件

```json
{
  "searchPackages": ["com.example"],
  "usingStrings": ["premium"],
  "returnType": "boolean",
  "paramTypes": ["int", "java.lang.String"],
  "paramCount": 2,
  "modifiers": "public",
  "methodName": "isVip",
  "className": "com.example.Vip",
  "usingNumbers": [1, 0.5, 114514]
}
```

`返回值`: string — JSON 数组

## dex.findField(query)

搜索字段。

`参数`:

`query`: string — JSON 查询条件

```json
{
  "searchPackages": ["com.example"],
  "fieldType": "int",
  "fieldName": "count",
  "modifiers": "private static",
  "className": "com.example.Counter"
}
```

`返回值`: string — JSON 数组

```json
[{"className":"com.example.Counter","name":"count","type":"int","accessFlags":10,"descriptor":"Lcom/example/Counter;->count:I"}]
```

## dex.batchFindClassByStrings(queriesJson)

批量按字符串搜索类（高性能，一次遍历完成多组搜索）。

`参数`:

`queriesJson`: string — JSON 格式 `{"key1":["str1","str2"], "key2":["str3"]}`

`返回值`: string — JSON 格式 `{"key1":[{classData},...], "key2":[{classData},...]}`

## dex.batchFindMethodByStrings(queriesJson)

批量按字符串搜索方法（高性能，一次遍历完成多组搜索）。

`参数`:

`queriesJson`: string — JSON 格式 `{"key1":["str1","str2"], "key2":["str3"]}`

`返回值`: string — JSON 格式 `{"key1":[{methodData},...], "key2":[{methodData},...]}`

## dex.getClassData(className)

获取类的详细信息，包括字段和方法列表。

`参数`:

`className`: string — 类全限定名，如 `"com.example.MyClass"`

`返回值`: string — JSON 格式

```json
{
  "name": "com.example.MyClass",
  "superClass": "java.lang.Object",
  "interfaces": ["java.io.Serializable"],
  "accessFlags": 1,
  "fields": [{"className":"com.example.MyClass","name":"id","type":"int","accessFlags":2,"descriptor":"..."}],
  "methods": [{"className":"com.example.MyClass","name":"getId","returnType":"int","paramTypes":[],"accessFlags":1,"descriptor":"..."}]
}
```

## 示例脚本

### 基本搜索

```javascript
// 初始化 DexKit (自动获取当前应用 APK)
dex.init(null);
console.log('DEX 文件数:', dex.getDexCount());

// 通过字符串搜索类 — 搜索包含 "VIP" 和 "expired" 的类
var classes = JSON.parse(dex.findClassByStrings('["VIP","expired"]', null, null));
for (var i = 0; i < classes.length; i++) {
    console.log('找到类:', classes[i].name);
}

// 释放资源
dex.close();
```

### 搜索方法并 Hook

```javascript
dex.init(null);

// 搜索包含 "premium" 字符串且返回 boolean 的方法
var methods = JSON.parse(dex.findMethod(JSON.stringify({
    usingStrings: ["premium"],
    returnType: "boolean",
    paramCount: 0
})));

for (var i = 0; i < methods.length; i++) {
    console.log('找到方法:', methods[i].className + '.' + methods[i].name);
    
    // 使用 Frida Hook 搜索到的方法
    Java.perform(function() {
        var cls = Java.use(methods[i].className);
        cls[methods[i].name].implementation = function() {
            console.log('方法被调用, 强制返回 true');
            return true;
        };
    });
}

dex.close();
```

### 批量搜索

```javascript
dex.init(null);

// 一次批量搜索多组字符串 (比多次调用 findClassByStrings 更高效)
var result = JSON.parse(dex.batchFindClassByStrings(JSON.stringify({
    "vip_class": ["VIP", "Premium"],
    "ad_class": ["AdMob", "Banner"],
    "pay_class": ["payment", "purchase"]
})));

for (var key in result) {
    console.log('[' + key + '] 找到 ' + result[key].length + ' 个类:');
    for (var i = 0; i < result[key].length; i++) {
        console.log('  - ' + result[key][i].name);
    }
}

dex.close();
```

### 获取类详细信息

```javascript
dex.init(null);

var info = JSON.parse(dex.getClassData('com.example.VipManager'));
if (info.name) {
    console.log('类名:', info.name);
    console.log('父类:', info.superClass);
    console.log('接口:', JSON.stringify(info.interfaces));
    console.log('方法数:', info.methods.length);
    for (var i = 0; i < info.methods.length; i++) {
        var m = info.methods[i];
        console.log('  方法:', m.name + '(' + m.paramTypes.join(',') + ') -> ' + m.returnType);
    }
}

dex.close();
```

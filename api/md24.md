# r2 — r2frida 逆向分析引擎

基于 [r2frida](https://github.com/nowsecure/r2frida) 项目，提供反汇编、内存搜索、模块分析、函数追踪、拦截等底层逆向工程能力。
所有方法均在 Frida GumJS 运行时中直接执行，无需 Java 层桥接。

---

## 反汇编

### `r2.disasm(address, count)`

反汇编指定地址的机器码指令。

| 参数 | 类型 | 说明 |
|------|------|------|
| address | `string` / `NativePointer` | 起始地址 |
| count | `number` | 指令数量，默认 32 |

**返回**: `string` — 反汇编文本，包含地址、助记符、操作数和符号信息。

```javascript
var base = Module.findBaseAddress("libil2cpp.so");
var asm = r2.disasm(base.add(0x1234), 10);
console.log(asm);
// ;;; libil2cpp.so
// 0x71a2b234  mov  x0, x1
// 0x71a2b238  bl   #0x71a2c000
// ...
```

## 内存

### `r2.hexdump(address, size)`

以十六进制格式转储内存内容。

| 参数 | 类型 | 说明 |
|------|------|------|
| address | `string` / `NativePointer` | 起始地址 |
| size | `number` | 字节数，默认 256 |

```javascript
r2.hexdump(base.add(0x1000), 64);
```

### `r2.memoryRanges()`

列出进程所有内存区域范围。

**返回**: `string` — 每行格式: `base - end protection [path]`

### `r2.memoryRangesJson()`

返回 JSON 格式的内存区域列表。

### `r2.memoryRangesHere(address)`

查找指定地址所在的内存区域。

### `r2.memoryMaps()`

列出内存映射（合并连续同路径区域）。

### `r2.allocate(size)`

在堆上分配指定大小的内存块。

| 参数 | 类型 | 说明 |
|------|------|------|
| size | `number` | 字节数 |

**返回**: `string` — 分配的地址

### `r2.allocateString(str)`

在堆上分配 UTF-8 字符串。

**返回**: `string` — 字符串地址

### `r2.freeAlloc(addr)`

释放之前通过 `allocate` / `allocateString` 分配的内存。

### `r2.listAllocs()`

列出所有通过 r2 分配的内存地址。

### `r2.changeProtection(addr, size, perm)`

修改内存区域的保护属性。

| 参数 | 类型 | 说明 |
|------|------|------|
| addr | `string` | 内存地址 |
| size | `number` | 大小 |
| perm | `string` | 权限字符串，如 `"rwx"`、`"r-x"` |

## 搜索

### `r2.searchString(pattern)`

在可读内存范围中搜索字符串。

| 参数 | 类型 | 说明 |
|------|------|------|
| pattern | `string` | 要搜索的字符串 |

**返回**: `string` (JSON) — 命中列表 `[{address, size}]`

```javascript
var hits = JSON.parse(r2.searchString("GameManager"));
hits.forEach(function(h) {
    console.log("Found at: " + h.address);
});
```

### `r2.searchHex(hex)`

搜索十六进制模式。

```javascript
var hits = JSON.parse(r2.searchHex("48 89 5c 24 08"));
```

### `r2.searchValue4(value)`

搜索 4 字节（32位小端序）数值。

```javascript
// 搜索金币值 999
var hits = JSON.parse(r2.searchValue4(999));
```

### `r2.searchValue8(value)`

搜索 8 字节（64位小端序）数值。

### `r2.searchWide(pattern)`

搜索 UTF-16 宽字符串。

### `r2.searchStrings(minLen)`

扫描内存中的所有可打印字符串。

| 参数 | 类型 | 说明 |
|------|------|------|
| minLen | `number` | 最小长度，默认 10 |

**返回**: `string` (JSON) — `[{address, text}]`

## 模块 / 符号

### `r2.modules()`

列出所有已加载的模块。

**返回**: `string` — 每行: `base size name path`

### `r2.modulesJson()`

返回 JSON 格式的模块列表。

### `r2.symbols(moduleName)`

列出指定模块的符号。

```javascript
r2.symbols("libil2cpp.so");
```

### `r2.exports(moduleName)`

列出指定模块的导出函数。

```javascript
r2.exports("libc.so");
```

### `r2.imports(moduleName)`

列出指定模块的导入函数。

### `r2.lookupAddress(address)`

查找地址对应的符号名和模块。

```javascript
var info = r2.lookupAddress(base.add(0x5678));
console.log(info); // "libil2cpp.so il2cpp_init 0x71a2b5678"
```

### `r2.lookupExport(name, moduleName)`

查找导出函数的地址。`moduleName` 可选，不传则搜索所有模块。

```javascript
var addr = r2.lookupExport("il2cpp_string_new", "libil2cpp.so");
```

### `r2.lookupSymbol(name, moduleName)`

查找符号的地址。

## 类枚举 (Java)

### `r2.classes()`

列出所有已加载的 Java 类。

### `r2.classMethods(className)`

列出指定 Java 类的所有方法。

```javascript
var methods = r2.classMethods("com.example.GameActivity");
console.log(methods);
```

### `r2.classFields(className)`

列出指定 Java 类的所有字段。

### `r2.classLoaders()`

列出所有已实例化的 ClassLoader。

## 追踪

### `r2.trace(address)`

追踪函数调用。自动打印调用栈、参数和返回值。

支持前缀 `java:` 追踪 Java 方法。

```javascript
// 追踪 native 函数
r2.trace(base.add(0x1234));

// 追踪 Java 方法
r2.trace("java:com.example.App.checkLicense");
```

### `r2.listTraces()`

列出当前所有活跃的追踪点。

### `r2.clearTrace(address)`

清除指定地址的追踪。

### `r2.clearAllTraces()`

清除所有追踪。

---

## 拦截

### `r2.interceptRet0(target)`

拦截函数并强制返回 0。支持 `java:` 前缀。

```javascript
r2.interceptRet0(base.add(0x5678));
r2.interceptRet0("java:com.example.App.isVip");
```

### `r2.interceptRet1(target)`

拦截函数并强制返回 1。

### `r2.interceptRetInt(target, value)`

拦截函数并强制返回指定整数值。

### `r2.interceptRetString(target, str)`

拦截函数并强制返回指定字符串。

### `r2.interceptDetachAll()`

清除所有拦截。

---

## Stalker

### `r2.stalkFunction(address, timeout)`

使用 Frida Stalker 进行指令级函数追踪 (实验性)。

| 参数 | 类型 | 说明 |
|------|------|------|
| address | `string` | 函数地址 |
| timeout | `number` | 超时秒数，默认 10 |

**返回**: `Promise<string>` — 调用汇总（次数 × 符号名）

```javascript
r2.stalkFunction(base.add(0x1234), 5).then(function(result) {
    console.log(result);
});
```

## 通用

### `r2.info()`

返回进程信息的 JSON 字符串（PID、架构、平台等）。

### `r2.version()`

返回 r2frida 版本号。

### `r2.cmd(command)`

执行 r2 风格的命令字符串。适合熟悉 radare2 的高级用户。

```javascript
r2.cmd("pd 0x71a2b234 20");    // 反汇编
r2.cmd("il");                   // 模块列表
r2.cmd("/x 48 89 5c 24 08");   // 十六进制搜索
r2.cmd("dm");                   // 内存映射
r2.cmd("dt 0x71a2b234");       // 追踪
r2.cmd("di0 0x71a2b234");      // 拦截返回0
r2.cmd("ic");                   // Java类列表
```

支持的命令列表: `pd`, `px`, `dm`, `dmj`, `dm.`, `dmm`, `dma`, `dmas`, `dma-`, `dmal`, `dmp`, `/`, `/x`, `/v4`, `/v8`, `/w`, `/z`, `il`, `ilj`, `is`, `iE`, `ii`, `fd`, `iaE`, `isa`, `ic`, `icm`, `icf`, `icL`, `dt`, `dt-`, `dt-*`, `di0`, `di1`, `dii`, `dis`, `di-1`, `di-*`, `dr`, `dpt`, `dptj`, `ls`, `cat`, `dd`, `env`, `dl`, `dp`, `i`, `?V`

## 完整示例

### 示例 1: 分析 native 库

```javascript
// 找到模块基址
var base = Module.findBaseAddress("libgame.so");

// 反汇编入口点
console.log(r2.disasm(base, 20));

// 列出导出
console.log(r2.exports("libgame.so"));

// 搜索字符串
var hits = JSON.parse(r2.searchString("LevelManager"));
console.log("Found " + hits.length + " hits");
```

### 示例 2: 内存值搜索与修改

```javascript
// 搜索金币值
var hits = JSON.parse(r2.searchValue4(9999));
console.log("Found " + hits.length + " addresses");

// hexdump 查看
hits.forEach(function(h) {
    console.log(r2.hexdump(h.address, 32));
});
```

### 示例 3: Java 方法分析

```javascript
// 列出指定类的方法
console.log(r2.classMethods("com.example.GameActivity"));

// 追踪方法
r2.trace("java:com.example.GameActivity.getScore");

// 拦截方法返回值
r2.interceptRet1("java:com.example.GameActivity.isPremium");
```

### 示例 4: r2 命令模式

```javascript
// 使用 r2 命令风格
console.log(r2.cmd("il"));          // 模块列表
console.log(r2.cmd("is libc.so"));  // libc 符号
console.log(r2.cmd("dm"));          // 内存映射
console.log(r2.cmd("dpt"));         // 线程列表
console.log(r2.cmd("ic"));          // Java 类列表
```
